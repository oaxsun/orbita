# DRKPRTY — Firebase Storage Setup

Para que el campo "Subir imagen a Firebase Storage" funcione:

1. Entra a Firebase Console.
2. Ve a Build > Storage.
3. Crea el bucket si aún no existe.
4. Ve a Rules.
5. Pega el contenido de `storage.rules`.
6. Publica las reglas.

Después:
- En Admin > Artículos puedes elegir una imagen desde tu computadora.
- Al guardar, el admin sube la imagen a Firebase Storage.
- La URL pública se guarda automáticamente en el campo `image` del artículo en Firestore.

Notas:
- Las imágenes se guardan en la carpeta `articles/`.
- El límite configurado es 10 MB por imagen.
- Para SEO/social preview, después podemos agregar campos `imageAlt`, `seoTitle`, `seoDescription` y meta tags dinámicos.
